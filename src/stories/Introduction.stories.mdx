# Welcome to Polaris Vue

Polaris Vue by HulkApps is a component library for [Vue.js](https://vuejs.org/) based on
[Shopify Polaris style guide](https://polaris.shopify.com/).

## Quick start

You need [Vue.js](https://vuejs.org/) **version 2+**.

### 1. Install via npm

```bash
npm i @hulkapps/polaris-vue
```

### 2. Import and use Polaris Vue

```javascript
import Vue from 'vue';
import PolarisVue from '@hulkapps/polaris-vue';
import '@hulkapps/polaris-vue/dist/polaris-vue.css';

Vue.use(PolarisVue);
```
