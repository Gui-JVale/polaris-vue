PList example:

Bullet list
```vue
  <PList>
    <PListItem>Item 1</PListItem>
    <PListItem>Item 2</PListItem>
    <PListItem>Item 3</PListItem>
    <PListItem>Item 4</PListItem>
  </PList>
```

Number list
```vue
  <PList type="number">
    <PListItem>Item 1</PListItem>
    <PListItem>Item 2</PListItem>
    <PListItem>Item 3</PListItem>
    <PListItem>Item 4</PListItem>
  </PList>
```
